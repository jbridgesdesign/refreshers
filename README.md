# 🍓 We Love Refreshers — Tribute Page

A fun, colorful tribute page dedicated to Starbucks Refreshers, built as a personal HTML & CSS practice project.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

---

## 📖 About

This is a self-directed project to test and reinforce my foundational knowledge of **HTML** and **CSS**. The subject? My favorite Starbucks drinks — Refreshers! The page is bright, vibrant, and full of personality, just like the drinks themselves.

---

## ✨ Features

- **Flavor Guide** — A curated list of current Starbucks Refreshers flavors with direct links to the Starbucks menu
- **Flavor Alterations** — A rundown of creative mix-ins and variations (think: coconut milk, lemonade, chili powder, and more)
- **Favorites List** — Personal favorites including retired and seasonal flavors
- **Release Timeline** — A sourced history of Refreshers from 2012 to present
- **Responsive Layout** — Clean structure with navigation, main content sections, and a footer
- **Custom Typography** — Google Fonts integration (*Oi* + *Work Sans*)

---

## 🛠️ Built With

- HTML5 (semantic elements: `header`, `main`, `section`, `figure`, `footer`, `nav`)
- CSS3 (custom stylesheet with Google Fonts)

## 🚀 Getting Started

No build tools or dependencies required — just open it in a browser!

1. Clone the repository:
   ```bash
   git clone https://github.com/jbridgesdesign/refreshers.git
   ```
2. Open `index.html` in your browser.

---

## 📚 What I Practiced

- Semantic HTML structure
- Navigation with anchor links
- Linking to external resources (with `target="_blank"`)
- Using `<figure>` and `<figcaption>` for images
- Organizing content with nested `<section>` elements
- Footnote-style source citations with internal anchor links
- Integrating Google Fonts via `<link>`
